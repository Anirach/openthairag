Title: จุดชมวิวสุดแผ่นดิน (Sud Pandin View Point)
Content: จุดชมวิวสุดแผ่นดิน อุทยานแห่งชาติป่าหินงาม อำเภอเทพสถิต จังหวัดชัยภูมิ 

เป็น จุดชมวิว ทะเลหมอก + พระอาทิตย์ตก 

ณ จุดนี้เป็นจุดที่สูงที่สุดของเทือกเขาพังเหย ( 846 เมตร )
เกิดจากการดันตัวของแผ่นดินภาคกลาง ( ฉานไทย )
ซุกเข้าใต้แผ่นดินภาคอีสาน ( อินโด ไชน่า )
ทำให้เกิดขอบยกสูงขึ้น แบ่งระหว่างภาคกลางกับภาคอีสาน
จึงเรียกบริเวณนี้ว่า สุดแผ่นดินอีสาน กลาง

Nearby Location: อยู่ในอุทยานแห่งชาติป่าหินงาม ห่างจากที่ทำการอุทยานประมาณ 2 กิโลเมตร
